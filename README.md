## 💻 About This Repository (My Portfolio)

This repository holds the source code for my personal portfolio website. 

### Features Include:
- Responsive, futuristic design with custom **Splash Screen** and smooth interactions.
- Dark/Light Theme toggle functionality powered by localized state & custom CSS variables.
- Interactive **Particles Background** and sleek micro-animations.
- Built using **React + Vite** for blazing-fast HMR and performance.

### Running it locally
1. Clone the repo: `git clone https://github.com/HipScout/Portfolio.git`
2. Install dependencies: `npm install`
3. Hit running: `npm run dev`
